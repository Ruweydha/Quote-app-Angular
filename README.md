# Quotes
#### A Webpage for publishing, viewing and voting for quotes.
#### By Ruweydha Abdinoor.
## Description
  This is an application that allows a user to publish a quote by filling out a form. The user can then see their quote and the quote can be liked or disliked. The quote with the highest likes is higlighted. The user can also delete a quote.
## Setup/Installation 
You can fork this repo and clone it to your device using git clone command and pasting your url after forking.

You can also acces it through a web browser no installation is required.

## Requirements
* Either a computer,phone,tablet or an Ipad
* An access to the Internet

## BDD.
* The user publishes a quote by filling out a form.
* The user can view the quote.
* The user can view the details of the quote and when it was published.
* The user can vote for a quote and even delete a quote.
* Quote with highest votes is highlighted.
